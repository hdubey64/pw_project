Static Position:-

The static position is the default position for all HTML elements. In the static position, elements follow the normal flow of the document.

This is a static positioned element.

Fixed Position:-

The fixed position keeps the element in a fixed position relative to the browser window. It remains in the same position even if the page is scrolled.

This is a fixed positioned element.

Sticky Position:-

The sticky position is a mix of both the relative and fixed positions. It starts as a normal element, but when the user scrolls to a specific threshold, it becomes fixed.

This is a sticky positioned element.

Relative Position:-

The relative position moves the element relative to its normal position without affecting the layout of other elements. It allows you to position an element based on its original position.

This is a relatively positioned element.

Absolute Position:-

The absolute position removes the element from the normal flow of the document and positions it relative to its closest positioned ancestor (if any) or to the initial containing block.

This is an absolutely positioned element.
